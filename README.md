## About

`cephcmd` 按阿里osscmd协议封装，兼容osscmd的使用语法，使用python语言开发。

## Install

```
直接下载，需要python2.7x环境
```

## Dependency

```
1、boto
2、FileChunkIO
```

## Usage

```shell

python cephcmd --help



```
